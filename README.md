# text-editor

## Description

Module 19 - PWA Progressive Web Aplications - Text Editor

This application helps you to create notes or code snipets with or without internet connection so you can easily retrieve them for later use.

## Built with:
- Javascript
- Node.js
- Express.js
- Concurrently
- Webpack
- Workbox
